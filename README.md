# 🧮 Calculadora Web Interativa

Uma aplicação web de calculadora simples e totalmente funcional desenvolvida com tecnologias web nativas, referente a disciplina Bootcamp. O projeto apresenta uma interface responsiva e centralizada com um esquema de cores moderno em tons de roxo e magenta, ideal para estudos de manipulação do DOM (Document Object Model) com JavaScript.

---

## 🚀 Demonstração Visual e Recursos

* **✨ Design Moderno:** Cores customizadas (`rgb`) com efeitos de transição visual (*hover*) nos botões.
* **📱 Layout Centralizado:** Interface perfeitamente posicionada no centro da tela usando propriedades avançadas de CSS (`transform` e `position: absolute`).
* **⚙️ Funções Operacionais:**
  * **`C` (Clean):** Limpa todo o visor da calculadora instantaneamente.
  * **`<` (Back):** Apaga o último caractere digitado (sistema de *backspace*).
  * **Operações Básicas:** Multiplicação, divisão, soma, subtração e controle de casas decimais.
  * **Processamento Dinâmico:** Motor de cálculo em tempo real integrado ao botão de igual (`=`).
